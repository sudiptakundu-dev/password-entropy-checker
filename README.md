# Password Entropy Checker

**Author:** Sudipta Kundu  

**Description:**  
Password Entropy Checker is a frontend tool that evaluates the strength of passwords entered by the user. It provides a score and suggestions to help users create stronger, more secure passwords. This project demonstrates basic security awareness and algorithmic thinking in web development.  

**Technologies Used:**  
- HTML  
- CSS  
- JavaScript  

**Instructions to Run:**  
1. Download or clone the repository.  
2. Open the `index.html` file in any web browser.  
3. Enter a password in the input field.  
4. View the entropy score and recommendations for improvement.  

**Learning Objectives:**  
- Demonstrates understanding of password security principles and basic algorithms.  
- Provides practical frontend development experience.  
- Shows initiative in building functional tools; currently learning full-stack development.  

**Notes:**  
- Fully functional offline; no additional installations required.  
- Can be extended to integrate with backend validation in the future.
